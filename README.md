# Clubinho app
Idea of an all which is a mix of Twitter + Whatsapp

## Main parts

### Club
- Create club
- Delete club
- Add a friend to club
- Remove a friend from the club

### Friend
- Add a friend
- Remove a friend

### Post
- Create post in the club
- Edit post from the club
- Delete post from the club
- Clap post
- Comment post
